# **App Name**: Student Perks Hub

## Core Features:

- Benefit Loading (Admin): Admin interface for loading new student benefits, shows, events, and discounts into the database.
- Landing Page Management: Modular, editable landing page to display student benefits.
- User Authentication: Secure user authentication system with biometric login and/or traditional username/password.
- Notifications & Alerts: Real-time notifications and alerts for new shows, events, and discounts.
- Personalized Recommendations: AI-powered tool which reasons on the basis of user's historical actions to create personalized suggestions for relevant perks, discounts and events.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke feelings of aspiration, imagination, and knowledge. 
- Background color: Light purple (#F3E5F5), visibly related in hue, but desaturated to around 20% and very light.
- Accent color: Indigo (#3F51B5) to contrast with the primary, and draw the user's eye toward important features.
- Body and headline font: 'PT Sans', a humanist sans-serif providing both a modern and slightly warm feeling, and also high readibility.
- Use clean, modern icons to represent different benefit categories (e.g., discounts, events, shows).
- Modular layout for easy content updates and rearrangement of landing page elements.
- Subtle animations for loading new benefits or displaying alerts to enhance user experience.
